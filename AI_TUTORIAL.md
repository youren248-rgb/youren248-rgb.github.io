# AI 视觉创作入门教程

这是一套给个人网站、封面图、作品集视觉使用的 AI 创作流程。它不追求一次生成完美结果，而是把想法拆成可以执行、可以复盘、可以交付的步骤。

## 适合谁

- 想用 AI 做网站首屏图、海报、封面、社交媒体配图的人
- 想把模糊灵感变成清晰视觉方向的人
- 想建立一套稳定创作流程，而不是每次随机试图的人

## 你需要准备

- 一个明确主题，例如“个人视觉作品集首页”
- 一个图片生成工具
- 一个文字 AI 工具，用来整理需求和提示词
- 一个图片压缩或裁切工具
- 一个用于记录版本的文件夹

## 总流程

1. 定义任务
2. 拆解关键词
3. 生成第一轮方向
4. 筛选和复盘
5. 精修和适配
6. 发布到网站或内容模板

## 01. 定义任务

开始之前，先不要写提示词。先用一句话说明你到底要什么。

示例：

```text
我需要一张用于个人作品集首页的视觉图，主题是 3D 动态、粒子系统和 AI 创作，画面要有科技感、空间感和情绪张力，适合深色网页背景。
```

然后补充 5 个限制：

- 用途：首页首屏图
- 比例：16:9 或 21:9
- 风格：未来感、清爽、精致
- 不要：杂乱文字、真实品牌标志、过度赛博朋克
- 输出：PNG 或 WebP

## 02. 拆解关键词

把提示词拆成 7 个模块，后面每次修改只改其中一个模块。

```text
主体：抽象 3D 能量核心、粒子场、光线轨道
场景：深色空间、轻微雾气、远处网格
材质：玻璃、荧光、半透明体积光
光线：青绿色主光、蓝紫色边缘光
构图：中心聚焦、周围粒子向内流动、留出文字区域
情绪：安静、专注、未来感、好奇
限制：不要文字、不要人物、不要 logo
```

## 03. 生成第一轮方向

第一轮不要追求细节，目标是快速找到方向。建议一次生成 4 到 8 张。

可直接复制的提示词：

```text
An abstract 3D energy core floating in a dark spatial field, surrounded by flowing particle systems and soft orbital light trails, translucent glass material, cyan and violet rim light, subtle volumetric fog, elegant futuristic portfolio hero image, centered composition with empty space for typography, refined, calm, high detail, no text, no logo, no people
```

中文版本：

```text
深色空间中的抽象 3D 能量核心，周围有流动粒子系统和柔和光轨，半透明玻璃材质，青绿色与蓝紫色边缘光，轻微体积雾，适合作品集首页的未来感视觉图，中心聚焦，预留文字区域，精致、安静、高细节，不要文字，不要标志，不要人物
```

## 04. 筛选和复盘

不要只问“哪张好看”，而是按这 4 个问题判断：

- 是否能一眼看出主题？
- 是否能放在网站上，不抢正文信息？
- 颜色是否和网站统一？
- 手机端裁切后是否还成立？

建议建立一个简单记录：

```text
V01：氛围好，但主体太复杂
V02：构图稳定，可以继续
V03：颜色偏脏，放弃
V04：有空间感，但文字区域不够
```

## 05. 精修和适配

选中一个方向后，下一轮只改一个变量。

例如：

```text
Keep the same composition, reduce visual noise, make the center object cleaner, increase empty space on the left side for large typography, keep cyan and violet lighting, no text, no logo
```

中文版本：

```text
保持同样构图，减少画面噪点，让中心物体更干净，在左侧增加适合大标题的留白，保留青绿色和蓝紫色光线，不要文字，不要标志
```

导出时准备三个尺寸：

- 桌面端：1920 x 1080
- 宽屏首屏：2400 x 1200
- 移动端：1080 x 1440

## 06. 发布和复盘

放进网站前，检查：

- 文件名是否清楚，例如 `hero-visual-goodmood-v01.webp`
- 文件大小是否压缩到合理范围
- 图片在深色背景上是否清晰
- 文字是否被背景干扰
- 手机端是否裁切到关键主体

最后写一段复盘：

```text
这次视觉方向使用粒子向中心聚合的结构，表达“想象被编译为运动”。青绿色负责清爽和科技感，蓝紫色负责空间深度。下一版需要减少背景噪点，提高移动端识别度。
```

## 推荐练习

用这套流程做一张“个人网站首页视觉图”。要求：

- 主题和你自己的身份相关
- 不直接生成文字
- 至少保留 3 个失败版本
- 最终输出桌面端和手机端两个比例
- 写下 100 字以内的复盘

## 核心原则

AI 负责扩大可能性，人负责判断方向。好的 AI 创作不是把选择权交出去，而是更快地看到更多可能，然后更清醒地做决定。
